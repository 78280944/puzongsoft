【利达牛牛】游戏规则说明：

＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝

★以下所有投注皆含本金。

★本游戏是根据重庆时时彩、天津时时彩、新疆时时彩、云南时时彩、黑龙江

时时彩官方的开奖结果采集而来,保证游戏结果的公正性。

★每期重庆时时彩、天津时时彩、新疆时时彩、云南时时彩、黑龙江时时彩开

奖结果共有5个球，按开奖时间顺序定义为1号球～5号球，每个开奖球的号码

值范围为0～9。

★本游戏采集5个球中的其中3个球按照从左至右的顺序任意组合 (开奖结果

出来后转换按三张扑克牌)作为游戏比赛台,按照比赛台的组合结果(以下简称

组合)大小排列十门牌名次:第一名～第十名。

采集①②③号球为第一门牌(亦称1台)

采集①②④号球为第二门牌(亦称2台)

采集①②⑤号球为第三门牌(亦称3台)

采集①③④号球为第四门牌(亦称4台)

采集①④⑤号球为第五门牌(亦称5台)

采集③④⑤号球为第六门牌(亦称6台)

采集②④⑤号球为第七门牌(亦称7台)

采集②③⑤号球为第八门牌(亦称8台)

采集②③④号球为第九门牌(亦称9台)

采集①③⑤号球为第十门牌(亦称10台)

★开彩结果包含【豹子、顺子（倒顺）、对子、牛牛（牛零）～牛九】:

【豹子】组合出现数值相同的三个数值时(如1、1、1)，则对应比赛台的开彩结

果为豹子；

【牛牛（牛零）】组合总和数值为10的整倍数时（如10、20），则对应比赛台的

开彩结果为牛牛（牛零）；

【顺子】组合出现依次顺序递增数值时(如1、2、3)，则对应比赛台的开彩结果

为顺子；

【倒顺】组合出现依次顺序递减数值时(如3、2、1)，则对应比赛台的开彩结果

为倒顺；

【对子】组合出现数值相同的两个数值时（如1、3、1；2、2、6），则对应比赛

台的开彩结果为对子；

【牛一～牛九】根据组合总和数值的个位数值,分别将对应比赛台的开彩结果定

义为牛一～牛九(如组合总和数值为09，则对应比赛台的开彩结果为牛九;如组

合总和数值为16，则对应比赛台的开彩结果为牛六)。

★本游戏赢输的规则为：豹子﹥牛牛﹥倒顺﹥顺子﹥对子﹥牛九﹥牛八﹥牛七

﹥牛六﹥牛五﹥牛四﹥牛三﹥牛二﹥牛一。

⑴如果比赛台的开彩结果出现相同，按照组合总和值大的一方为赢(如组合总和

数值为10和20的客户，其对应比赛台的开彩结果均为牛牛（牛零），游戏结果

则是总和数值大的20的客户为赢家)。

⑵如果比赛台的开彩结果出现同样为对子的，按照对子数值大的一方为赢(如2、

2、6和5、5、3同样为对子，游戏结果则是对子数值大的5、5、3为赢家)。如

果比赛台的开彩结果出现同样为对子且对子数值相同的，按照组合总和值大的

一方为赢(如2、2、5和2、2、7同样为对子且对子数值同样为2，游戏结果则

是组合总和值大的2、2、7为赢家)。

⑶ 如果比赛台的开彩结果出现相同总和值的顺子和倒顺，按照倒顺一方为赢

(如顺子1、2、3和倒顺3、2、1，总和值相同，游戏结果则是倒顺3、2、1为

赢家) 。

⑷如果比赛台的开奖结果出现完全相同时(组合里的3个球的数值和顺序)，则

为和局。

★客户可以选择做“我要上庄”或是作为“闲家”自由选择比赛台参与游戏。

有庄无庄时都可以下注,若游戏房间不存在其他游戏台的客户与您产生输赢关

系,则结算后退款。

【无庄时】由所有的比赛台相互进行比赛,然后根据各个比赛台的游戏结果进行

输赢判断。

【有庄时】由庄台和其他所有的比赛台进行比赛,然后根据庄台的游戏结果和各

个比赛台的游戏结算比赛进行输赢判断(注:所有比赛台都是直接和庄台进行比

赛,庄台以外的其他比赛台相互间不另外进行比赛)。

★结算遵循原则:

【无庄时】

⑴先将第一名和第十名进行结算,如果第十名的投注金额不够赔付,则第一名继

续和第九名进行结算……直到结算完毕;如果按照此结算规

则直到第一名和第二名结算完毕后仍不够赔付第一名的投注额,则系统不再进

行结算,第一名的所有投注客户则根据所投注在第一名比赛台里金额所占比赛

台投注总金额的比例数作为结算依据,将所有赔付总额按比例进行分配结算,

分配结算规则按照:投注比例小的客户优先结算,比例小的结算完毕后再到投注

比例大的客户结算的原则进行分配结算,直到赔付总额分配完毕。

⑵在第一名得到足够的赔付金额后, 由第二名按照此结算规则继续进行结

算……直到完成所有比赛台的结算。

【有庄时】

(注:作庄的比赛台以下简称庄台;游戏结果小于庄台以下简称输台; 游戏结果

大于庄台以下简称赢台)

⑴庄台优先按照从小到大的结算规则和输台进行结算,直到结算完毕。如果按照

此结算规则庄台在和输台结算完毕后,已足够赔付庄台的投注额,则庄台即可按

照分配结算规则将所得到的赔付额进行本庄台所有投注者的分配结算,而不需

要再向赢台进行赔付结算,本轮游戏结算完毕 (注:赢台相互间不另外进行结

算,剩余的输台亦不需要再向庄台赔付) 。

⑵庄台按照从大到小的结算规则和赢台进行结算,直到结算完毕。如果按照此结

算规则庄台在和所有输台结算完毕后,仍不足以赔付庄台的投注额,则庄台需要

将之前和输台所得到的结算赔付额以及原来自己的投注额按照从大到小的赔付

规则向赢台进行结算赔付:

  1. 庄台和所有赢台结算后,仍有剩余额,则庄台可将剩余额进行本庄台所有投注

者的分配结算。

B. 庄台在和赢台结算时,不足够赔付赢台的投注额额, 则剩余的赢台亦不能够

再向庄台要求结算,本轮游戏结算完毕(注:剩余赢台相互间不另外进行结算)。
