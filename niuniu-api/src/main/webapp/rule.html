游戏规则说明：

＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝

★以下所有投注皆含本金

★本游戏开彩结果按照重庆快乐十分当期开彩结果的最后一码（简称开出之号码）减去最接近的4的倍数值后所剩余Q4的正数值（1、2、3、4）作为本游戏的开彩结果（简称开彩结果）。如：开出之号码为15，则开彩结果为3；开出之号码为12，则开彩结果为4。

★每期重庆快乐十分开奖结果共有8个球，选择号码为1号－20号。

★重庆快乐十分每天开84期（10：00－2：00），每期间隔10分钟，投注时间为8分钟，等开奖时间为2分钟。

【角】即下注XY，当开彩结果为X或者Y赢，开其他的为输。

【连】即X连Y，即下注X，当开彩结果为X赢，开彩结果为Y和局，

开其他的为输。

如：开彩结果为1时，下注1连2、1连4赢；下注2连1、4连1为和，开其他的为输。

【番】即下注X，当开彩结果为X赢，开其他的为输。

【正】即下注X，当开彩结果为X赢，开彩结果为对面输，开彩结果为两边和局。（1对3，2对4）

如：开彩结果为1时，即下注“一正”则赢，“三正”输，“二”以及“四正”为和。

【三中】即下注XYZ，当开出X、Y、Z其中一个为赢，开其他的为输。

【特码】投注号码为开出之号码，视为中奖，其余情况视为输。

【色】20个号码分为四种类，分别为黄、红、蓝、绿（如下），再依四种类下注。若开出之号码在相应的球位落在下注方位范围内，视为赢：

      黄：1、5、9、13、17

红：2、6、10、14、18

蓝：3、7、11、15、19

绿：4、8、12、16、20

【大/小】开出之号码R11为大，Q10为小。

【单/双】开出之号码为双数叫双，如：8、16；开出之号码为单数叫

单，如：9、15。
